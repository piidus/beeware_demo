# beeware_demo
This Package Developed using beeware library

-   Install
```bash
pip install briefcase
```

```bash
briefcase new
```

-   Create Bundle
```bash
briefcase create
```

-   Create exe
```bash
briefcase build
```
    -   check 
```bash 
briefcase run```

-   Create Installer
    `briefcase package`

-   Create Update
    `briefcase update`
    `briefcase run -u`

-   For Android
    `briefcase create android`
    `briefcase build android`
