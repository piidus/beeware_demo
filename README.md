# beeware_demo
This Package Developed using beeware library

-   Install
```bash
pip install briefcase
```

```bash
briefcase new
```

-   Create Bundle
```bash
briefcase create
```

-   Create exe
```bash
briefcase build
```
    -   check 
```bash 
briefcase run
```

-   Create Installer
```bash
briefcase package
```

-   Create Update
```bash
briefcase update
```
```bash
briefcase run -u
```

-   For Android
```bash
briefcase create android
```
```bash
briefcase build android
```
