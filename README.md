# beeware_demo
This Package Developed using beeware library

-   Install

`pip install briefcase`


`briefcase new`

-   Create Bundle
`briefcase create`

-   Create exe
    `briefcase build`
    -   check 
    `briefcase run`

-   Create Installer
    `briefcase package`

-   Create Update
    `briefcase update`
    `briefcase run -u`

-   For Android
    `briefcase create android`
    `briefcase build android`
